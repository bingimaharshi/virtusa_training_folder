# Resume Analyzer & Job Matcher

A Python NLP tool that reads resumes, extracts skills, and compares them against job descriptions to give a match score with improvement suggestions.

## Features

- Reads resumes in `.txt` or `.pdf` format
- Extracts email, phone, and technical skills automatically
- Compares skills against a job description
- Shows match percentage with a visual progress bar
- Lists matched and missing skills
- Provides actionable improvement suggestions
- Saves full analysis to `analysis_result.json`

## Tools Used

- **Python 3.x** - Core language
- **pdfplumber** - PDF text extraction
- **re (regex)** - Keyword and contact info extraction
- **json** - Result persistence
- **Pattern matching** - Skill category detection

## Skill Categories Detected

Programming Languages, Web Technologies, Databases, Cloud & DevOps, Data & AI, Soft Skills

## Setup & Run

```bash
pip install -r requirements.txt
python main.py
```
