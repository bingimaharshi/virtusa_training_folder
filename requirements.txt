# Weather Data Analyzer & Forecast Dashboard

A Python CLI application that fetches, stores, and analyzes weather data with visual trend charts and temperature prediction using linear regression.

## Features

- Fetch live weather via OpenWeatherMap API
- Manual weather data entry support
- Temperature and humidity trend visualization (dual-chart PNG)
- Simple linear regression for 1/3/7-day temperature prediction
- Offline mode with 14-day sample data for Hyderabad
- Historical data stored in `weather_history.json`

## Tools Used

- **Python 3.x** - Core language
- **urllib** - HTTP requests (no external HTTP dependency)
- **matplotlib** - Dual-panel trend chart (temperature + humidity)
- **statistics** - Mean, trend calculation
- **Linear Regression** - Built from scratch using slope/intercept formula
- **json** - Data persistence

## Setup

```bash
pip install -r requirements.txt

# Optional: for live weather data
export OPENWEATHER_API_KEY=your_key_here

python main.py
```
