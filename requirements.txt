# Smart Expense Tracker

A Python CLI application that helps you log, categorize, and analyze your daily expenses with visual insights.

## Features

- Add expenses with date, category, amount, and description
- View all recorded expenses in a clean table
- Monthly summary with category-wise breakdown
- Pie chart visualization saved as PNG
- Smart suggestions for reducing spending
- Delete entries when needed
- Data stored locally in `expenses.json`

## Tools Used

- **Python 3.x** - Core language
- **json** - Data persistence
- **matplotlib** - Pie chart visualization
- **collections.defaultdict** - Aggregation
- **datetime** - Date handling

## Setup & Run

```bash
pip install -r requirements.txt
python main.py
```

## Categories Supported

Food, Travel, Bills, Entertainment, Health, Shopping, Education, Other

## How It Works

1. Run the script and choose from the menu
2. Add daily expenses as they happen
3. At month end, select option 3 to see your summary
4. A pie chart is auto-generated and saved
